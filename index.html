    Tips

  • Bold Styling
    • Placing square bracketed open and closed [b] [/b] tags either side of text within the content text box will output bold text.

      Usage: [b] Hello [/b] Output:

      Hello

  • Please note:
    • Typing [note] within the content text box will generate a "Please note: ".

      Usage: [note] Output:

      Please note:

    • Usage

  • Add Section Button
    • Situated on the top tool bar, this generates a title and content text box in the left panel. Type/paste content into the fields to display live content on the right panel, and code in the bottom panel.
    • Delete Section Button
      • Generated on each section, this simply deletes the section.
  • Add Features Button
    • Situated on the top tool bar, this opens a feature panel. Type/paste features into the text box to generate a list of bullets that can be copied. Enter presses also generate a bullet on the list.
    • Copy Button
      • Copies the generated list of bullets.
    • Copy with Center Tags Button
      • Copies the generated list of bullets with inline open/close center tags at the beginning and end of the bulleted list.
  • Title Checker Button
    • Situated on the top tool bar, this opens a title checker panel. Type/paste features into the short and long text boxes to count the list of characters in the titles.

      This automatically removes multiple spaces, resulting in a well formatted title.
    • Clear Titles Button
      • Clears the titles from the text boxes, resetting the character count to 0.
    • Template Buttons
      • Produces a short and long template example of an existing product to encourage uniform titling.
  • Copy Code Button
    • Situated on the top tool bar, this copies the generated code with inline open/close center tags at the beginning and end of the code.
